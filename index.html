Sobre este Portal

Este portal apresenta dados oficiais sobre empresas sancionadas pelo Governo de Minas Gerais, em conformidade com a Lei Federal nº 12.846/2013 (Lei Anticorrupção).

Fonte dos Dados

Os dados são atualizados automaticamente a partir do conjunto de dados públicos disponível no Portal de Dados Abertos de Minas Gerais.
